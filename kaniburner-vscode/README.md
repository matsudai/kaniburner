# Kaniburner

Cross platform mruby compiler and mrb writer.

![](https://github.com/matsudai/kaniburner/blob/main/docs/images/readme.gif)

## Requirements

Firmware supports [mrbwrite protocol](https://github.com/mrubyc/mrbwrite).

## Getting Started

1. Open `.rb` file on VSCode.
2. Run `> kaniburner: Serve` on VSCode (and your default Web browser will open).
3. Connect your device (e.g. Raspberry Pi Pico) and click the connection button on the web page.
4. Select the device.
5. Run `> Kaniburner: Execute All` on VSCode.
